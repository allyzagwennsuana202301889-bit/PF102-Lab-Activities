## Reflection 
1. What does `await` do? Why do we need it?
- await is a pause for a promise or command to fulfill. A sentence example is “Don’t go off yet and wait until I get my purse.”

2. What happens if the API is down? How does our code handle that?
-When the API is down there’s a condition to implement like Try and Catch. Try is for fetching the data and the catch is when they cannot find the data or a specific condition is not met.
3. What's the difference between `try` and `catch`?
-Try is for fetching data or doing a specific condition while catch occurs whenever there’s errors. This is a blocking method that checks your code if something goes wrong or a specific condition is not met.
